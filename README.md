## 📋 Sobre o Projeto
Este é um aplicativo de fitness gamificado que transforma a evolução física do usuário em um sistema de RPG. Diferente de apps comuns, a interface e os treinos são **adaptados dinamicamente** através de algoritmos que analisam o peso e a altura do usuário.

## 🛠️ Funcionalidades
- **Análise Biométrica:** Cálculo inicial de atributos (FOR, AGI, VIT) baseado em dados reais.
- **Daily Quests Adaptáveis:** O sistema ajusta o volume de treino conforme seu biotipo.
- **Interface Holográfica:** UI inspirada no HUD de Solo Leveling.
- **Evolução de Status:** Ganho de XP e pontos de atributo ao concluir treinos.

## 🧬 Lógica de Adaptação (O Algoritmo)
O sistema utiliza a relação entre Peso e Altura para definir a dificuldade inicial:
- **Alto IMC:** Foco em queima calórica e resistência articular (Quest: HIIT/Caminhada).
- **Baixo IMC:** Foco em hipertrofia e calistenia de base (Quest: Flexões/Agachamentos).
